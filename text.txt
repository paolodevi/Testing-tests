Perfect — this is a plot-with-inset-zoom problem, not an image one. Below is the canonical MATLAB solution: full plot, a zoomed inset in the top-left, and a rectangle showing the zoomed region on the main axes.


---

Full plot + zoomed inset + highlighted zoom region

figure

% ---- Main axes ----
axMain = axes;
plot(x, y, 'LineWidth', 1.2)
grid on
hold on

% Define zoom region (DATA coordinates)
xZoom = [2.5 3.2];
yZoom = [0.4 0.8];

% Draw rectangle showing zoom area
rectangle('Position', ...
    [xZoom(1), yZoom(1), diff(xZoom), diff(yZoom)], ...
    'EdgeColor', 'r', ...
    'LineWidth', 1.5, ...
    'LineStyle', '--');

% ---- Inset axes (top-left corner) ----
axInset = axes('Position', [0.15 0.65 0.25 0.25]); % normalized figure units
box on
hold on

plot(x, y, 'LineWidth', 1.2)
xlim(xZoom)
ylim(yZoom)
grid on

% Optional: reduce clutter
set(axInset, 'FontSize', 8)

% Keep main axes active
axes(axMain)


---

Key points (important)

1. Rectangle is in data coordinates

Because this is a plot, rectangle uses axis data units, not pixels.

[x0 y0 width height]

This makes it robust if you rescale axes.


---

2. Inset position uses normalized figure coordinates

axes('Position', [left bottom width height])

Top-left corner ≈ [0.15 0.65 0.25 0.25]
Adjust as needed.


---

3. Same data plotted twice

The inset is just another axes plotting the same data, but with xlim/ylim set to the zoom region.


---

Optional: draw connecting lines (publication-quality)

annotation('line', [0.25 0.35], [0.65 0.45])
annotation('line', [0.40 0.55], [0.65 0.45])

(These use figure coordinates, so tweak visually.)


---

Optional: automatically link limits

If you want changes in zoom limits to propagate:

linkaxes([axMain axInset],'xy')

x = linspace(0,10,2000);
